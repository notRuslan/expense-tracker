# Standup Skill

Подготовь краткий standup-отчёт по тому, что было сделано вчера.

## Алгоритм выполнения

1. Определи вчерашнюю дату:
```bash
date -v-1d +%Y-%m-%d
```

2. Получи все коммиты за вчерашний день:
```bash
git log --after="<дата вчера> 00:00" --before="<дата вчера> 23:59" --format="%H %s"
```

3. Для каждого коммита получи diff:
```bash
git show <hash> --stat --patch
```

4. Проанализируй суть изменений и сформируй отчёт.

## Формат отчёта

Два коротких абзаца в свободной форме:

- **Первый абзац** — что было сделано: общая суть работы без перечисления файлов, в духе устного рассказа
- **Второй абзац** — контекст или детали, если есть что добавить; если нечего — один абзац

Не используй заголовки, списки, хэши коммитов. Пиши как для живого standup-митинга.

## Правила

- Описывай суть, а не файлы: не «изменён файл X», а «реализована авторизация через JWT»
- Если коммитов нет — одно предложение: «Вчера коммитов не найдено»
- Язык — русский

## Дополнительный режим: /standup <дата>

Если передана дата аргументом (например `/standup 2025-01-10`), используй её вместо вчерашней.
